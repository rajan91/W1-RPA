# UI Automation Demos 🚀

This repository contains hands-on automation examples using:

- **PyAutoGUI** – Desktop UI automation
- **Playwright** – Modern web automation

### 🔹 PyAutoGUI
- Automates EMI calculator using mouse, keyboard, and screen actions

### 🔹 Playwright
- Automates web EMI calculator using XPath selectors

## 🛠 Tools Used
- Python
- PyAutoGUI
- Playwright
